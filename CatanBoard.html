Guide


Premise: The idea of this tool is that you can randomly generate tiles and their number tokens according to a set format for a game of Catan.

If all you want is a basic game or the extension, simply select one of those maps at the top of the dropdown box near the bottom of the left hand side of the page. Once you've done that, click on the allot tiles button and you should have a randomly set board on the right side of the screen. You can do the exact same with all the other maps too.

If you'd like to create your own map, just select the kind of tile you want to place with the radio buttons on the left and click on the left hand map. For all tiles thereafter, they will need to be joined to the current board, so click beside any tile to add another. If you want to change a tile, click it again with a different tile. The 'random' tile will act just like the ones you've seen in the examples. Just be sure to provide enough in the tile supply to fill those random ones with. Also be sure to supply the kind of tokens you want to fill those tiles with, if they need tokens.

If your map becomes considerably large, you can move around by holding down shift and dragging with the mouse at the same time. Changing the scale (before you start a new map) will also make the tiles larger or smaller.
